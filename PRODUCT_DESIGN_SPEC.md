# 产品设计规范 — Vibe Coding 发现

## 1. 产品定位
GitHub 热门项目精选网站。用户打开首页就能看到各类别最火项目，点击分类可深入浏览。

## 2. 设计风格：Apple.com 风格

### 2.1 核心特征
- **极简留白**：大量白色/浅灰空间，内容呼吸感强
- **大标题+副标题**：每个区块有醒目标题（32-48px bold）
- **模块化滚动**：下滑浏览，每个模块独立完整
- **方块布局**：大方块（Hero）+ 小方块（项目卡片）交替排列
- **滚动动画**：元素进入视口时 fade-in + slide-up

### 2.2 配色系统
```
背景主色: #FFFFFF (白)
背景次色: #F5F5F7 (浅灰，苹果标准灰)
背景深色: #1D1D1F (深黑，用于深色区块)
文字主色: #1D1D1F (深黑)
文字次色: #86868B (灰色)
链接/按钮: #0071E3 (苹果蓝)
卡片边框: #D2D2D7
```

### 2.3 字体
```
标题: -apple-system, "SF Pro Display", "PingFang SC", sans-serif
正文: -apple-system, "SF Pro Text", "PingFang SC", sans-serif
标题字重: 600 (Semibold)
正文字重: 400 (Regular)
```

### 2.4 间距系统
```
区块间距: 80px (大区块之间)
卡片间距: 16px
内容最大宽度: 1200px
内边距: 24px (移动端) / 48px (桌面端)
```

### 3.2 分类详情页 (/category/[id])
- 点击分类标题或"查看全部"按钮跳转
- 展示该分类下所有项目（网格布局，每行4个）
- 支持排序（最多星标 / 最近更新）

### 3.3 导航栏
- 固定顶部，毛玻璃效果 (backdrop-blur)
- Logo 左侧，分类链接居中，搜索图标右侧
- 高度: 48px
- 背景: rgba(255,255,255,0.8) + backdrop-blur

## 4. 组件规格

### 4.1 分类区块 (CategorySection)
```
- 高度: auto (根据内容)
- 背景: 交替使用 #FFFFFF 和 #F5F5F7
- 内边距: 80px 上下, 48px 左右
- 标题: 32px, font-weight 600, color #1D1D1F
- 副标题: 18px, font-weight 400, color #86868B
- "查看全部" 链接: 蓝色 #0071E3, 右箭头
```

### 4.2 项目卡片 (ProjectCard) — 苹果风格
```
- 宽度: 等分排列，间距 16px
- 高度: 固定 280px
- 背景: #FFFFFF (白底区块) 或 #FFFFFF (灰底区块)
- 圆角: 18px
- 边框: 1px solid #D2D2D7
- 阴影: 无 (苹果风格不用阴影)
- Hover: scale(1.02) + 轻微阴影

内部布局:
┌────────────────────────┐
│                        │
│   项目图标/占位图       │  ← 80x80, 居中
│                        │
├────────────────────────┤
│  项目名称 (16px bold)  │  ← 单行截断
│  描述 (14px gray)      │  ← 2行截断
├────────────────────────┤
│  ★ 515k  TypeScript    │  ← 底部信息栏
└────────────────────────┘

内边距: 24px
```

### 4.3 Hero 区块
```
- 高度: 80vh (最小 600px)
- 背景: #FFFFFF
- 标题: 56px, font-weight 700, letter-spacing -0.5px
- 副标题: 24px, font-weight 400, color #86868B
- 搜索框: 最大宽度 600px, 高度 56px, 圆角 980px (胶囊形)
- 搜索框背景: #F5F5F7, 无边框
- 搜索框聚焦: 白色背景 + 蓝色边框
```

### 4.4 搜索框
```
- 高度: 56px
- 圆角: 980px (完全胶囊形)
- 背景: #F5F5F7
- 左侧: 搜索图标 (灰色)
- 占位文字: "搜索项目，如：记账、简历、AI..."
- 字体: 17px
- 聚焦效果: 背景变白 + box-shadow: 0 0 0 4px rgba(0,113,227,0.2)
```

## 5. 滚动动画
```
- 使用 Intersection Observer API
- 触发条件: 元素进入视口 20% 时
- 动画: opacity 0→1 + translateY(30px→0)
- 时长: 0.6s
- 缓动: cubic-bezier(0.25, 0.46, 0.45, 0.94)
- 每个卡片有 0.1s 的递增延迟 (stagger)
```

## 6. 分类模块（6个维度）
1. AI工具 — 最火 AI 开源项目
2. 效率工具 — 提升工作效率的神器
3. SaaS — 企业级 SaaS 模板
4. 个人网站 — 漂亮的个人网站/博客
5. 游戏 — 游戏引擎和游戏项目
6. 电商 — 电商和支付解决方案

## 7. 禁止清单
- 禁止 emoji（任何场景）
- 禁止彩色渐变背景
- 禁止圆角超过 18px（除了搜索框 980px）
- 禁止阴影过重（hover 阴影用 rgba(0,0,0,0.04)）
- 禁止花哨动画（只允许 fade-in + slide-up）
- 禁止深色模式（只做浅色）
- 禁止卡片内用图标代替文字标签

## 8. 响应式
```
桌面 (>=1200px): 每行 5 个卡片
平板 (>=768px): 每行 3 个卡片
手机 (<768px): 每行 1 个卡片
```

## 9. 技术约束
- Next.js 15 App Router
- Tailwind CSS 4
- 纯 CSS 动画（不用 framer-motion）
- GitHub API 数据（已有 lib/github.ts）
- 服务端渲染（SSR）
