## 一、产品定位

**目标用户：** 想做项目但不懂技术的小白
- 不懂英文，看不懂GitHub
- 想知道"这个项目能帮我做什么"
- 需要中文解读，需要手把手指导

**核心价值：** 筛选+解读+引导
- 筛选：从海量项目中挑出最适合小白的
- 解读：用通俗中文解释项目是什么、能做什么
- 引导：告诉小白怎么用（一键复制prompt给AI）

#### 2.2.2 内容生成逻辑

**数据来源：**
1. GitHub API 获取项目基本信息（名称、星标、语言）
2. GitHub API 获取 README.md（项目说明文档）
3. DashScope API（qwen3.7-plus）翻译+解读 README

**AI解读模板：**

```markdown
你是一个技术翻译专家，帮不懂技术的普通人解读GitHub项目。

请根据以下GitHub项目信息，生成通俗易懂的中文解读：

项目信息：
- 名称：{name}
- 描述：{description}
- 语言：{language}
- 星标：{stars}
- README内容：{readme_content}

请输出以下内容（用通俗中文，不用技术术语）：

## 这是什么？
（用1-2句话解释这个项目是什么，举生活中的例子）

## 适合做什么？
✅ 适合：
- （列出3-5个具体场景，用"想做XXX"的句式）

❌ 不适合：
- （列出2-3个不适合的场景，告诉用户应该用什么替代）

## 难度评估
（用⭐表示，1-5星）
- 技术门槛：⭐⭐⭐（需要懂JavaScript）
- 学习时间：⭐⭐（1-2周可以上手）
- 部署难度：⭐⭐（需要服务器，但有教程）

## 使用建议
（给小白的具体建议，告诉他们第一步做什么）
```

**生成策略：**
- 构建时生成（SSG），不每次请求都调AI
- 缓存到本地JSON文件（data/project-interpretations/[id].json）
- 每小时更新一次（revalidate: 3600）

## 五、技术实现

### 5.1 新增文件

```
app/
  project/
    [id]/
      page.tsx          # 项目详情页
      loading.tsx       # 加载状态
components/
  RankingList.tsx       # 排行榜列表组件
  RankingItem.tsx       # 排行榜单项组件
  ProjectInterpretation.tsx  # 项目解读组件
lib/
  interpretation.ts     # AI解读生成逻辑
  readme.ts            # README获取+解析
data/
  project-interpretations/  # 缓存目录
    [id].json
```

### 5.2 API调用

**获取README：**
```typescript
// lib/readme.ts
export async function fetchReadme(fullName: string): Promise<string> {
  const response = await fetch(
    `https://api.github.com/repos/${fullName}/readme`,
    {
      headers: { Accept: 'application/vnd.github.v3.raw' },
      next: { revalidate: 3600 },
    }
  );
  if (!response.ok) return '';
  return response.text();
}
```

**AI解读：**
```typescript
// lib/interpretation.ts
export async function generateInterpretation(project: Project, readme: string) {
  const prompt = `...`; // 见上方模板
  
  const response = await fetch('https://dashscope.aliyuncs.com/...', {
    method: 'POST',
    body: JSON.stringify({
      model: 'qwen3.7-plus',
      input: { messages: [{ role: 'user', content: prompt }] },
    }),
  });
  
  const data = await response.json();
  return data.output.text;
}
```

### 5.3 数据流

```
构建时（SSG）：
1. fetchTrendingRepos() → 获取Top 100项目
2. 取前20个 → 生成排行榜
3. 对每个项目：
   a. fetchReadme() → 获取README
   b. generateInterpretation() → AI生成解读
   c. 保存到 data/project-interpretations/[id].json
4. 生成 /project/[id] 页面（读取缓存的解读）

运行时：
- 首页：读取排行榜数据（已缓存）
- 详情页：读取解读缓存（已生成）
```
